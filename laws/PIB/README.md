# DFSA PIB Rulebook — Interactive Visual Guide

A comprehensive single-page interactive guide to the **DFSA Prudential – Investment, Insurance Intermediation and Banking (PIB) Module** (VER52/01-26).

## 🚀 Hosting on GitHub Pages

1. Create a new GitHub repository (e.g. `dfsa-pib-guide`)
2. Upload **both** files to the repository root:
   - `index.html` — the interactive website
   - `DFSA_PIB_Rulebook.pdf` — the source rulebook (required for "View Original" links)
3. Go to **Settings → Pages → Source → Deploy from branch → `main` / `(root)`**
4. Your site will be live at: `https://yourusername.github.io/dfsa-pib-guide/`

> Both files **must be in the same folder** for the PDF source links to work correctly.
